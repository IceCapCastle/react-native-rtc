###Android 配置
```
repositories {
  maven {
    url  "https://dl.bintray.com/arthasanddk/maven"
  }
}

compile 'com.arthas.rtc:react-native-rtc:1.0.2'
```

###iOS 配置
```
pod 'react-native-rtc', '1.0.2'

Header Search Paths $(SRCROOT)/../../../ios/Pods/Headers recursive
```
